# 🐾 Dogopedia 🐕 - A Dog Breeds Encyclopedia 🌍

Welcome to **Dogopedia**, your go-to place for learning about various dog breeds! 🐶 This web app provides information about different dog breeds, including their characteristics, behavior, and much more, through an easy-to-use interface. It pulls data from the [Dog API](https://thedogapi.com/) and displays a detailed list of breeds with images. 🖼️

🚀 **Live Demo**  
Check out the live version of the app here: [Dogopedia on Render](https://dogopedia-o6iw.onrender.com) 🌐

---

## Features 📝

- 🔍 Search for dog breeds
- 🖼️ View images of different breeds
- 🌍 Access detailed information on each breed
- 💻 Built with Node.js and Express for the backend
- 🔑 API key management for fetching data from The Dog API 🐕

---

## 🛠️ Technologies Used

- **Frontend**: 
  - HTML, CSS, JavaScript
- **Backend**:
  - Node.js
  - Express
  - Fetch API
  - dotenv for environment variables
- **API**: The Dog API [TheDogAPI.com](https://thedogapi.com/)
- **Deployment**: Render 🌐

---

## 💡 How It Works

1. The app uses **Node.js** and **Express** for the backend.
2. It fetches breed data from [The Dog API](https://thedogapi.com/).
3. Data is displayed on the frontend, including breed images and detailed information.
4. The API is securely integrated using **GitHub Secrets** for managing sensitive information like API keys.

## 🔧 Setup and Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Arpita23r/dogopedia.git
   ```

2. Navigate to the project folder:
   ```bash
   cd dogopedia
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Create a `.env` file in the root directory and add your **API_KEY**:
   ```bash
   API_KEY=your_api_key_here
   ```

5. Start the application:
   ```bash
   npm start
   ```

6. The app will run on `http://localhost:5000`. Open it in your browser! 🌐
